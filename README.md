# Schedule App

A modern scheduling and appointment booking application built with React, Vite, and Supabase.

## Features

- User authentication
- Appointment booking
- Provider management
- Admin dashboard
- Responsive design

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your environment variables in `.env`
4. Run the development server: `npm run dev`

## Technologies Used

- React
- Vite
- Tailwind CSS
- Supabase
- Shadcn/ui
